태그

    h1~h6 = 제목테그(숫자별 크기)
    strong = 굵게 강조
    underline(U) = 밑줄 치기
    p = 문단 나눔
    br = 줄바꿈태그
    img src = 이미지태그
    width = 크기 조절 %
    li = 목차태그
    ul = li부모태그
    ol = 자동번호매김
    table = 테이블(표만들기)
    tr = 테이블 안에 쓰는 부모태그
    td = 테이블 안에 쓰는 자녀태그
    a href = 링크달기
    target="_blank" = 새탭열기
